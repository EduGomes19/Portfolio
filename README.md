# Portfólio Dev.Eduu 🚀

Bem-vindo ao repositório do meu portfólio pessoal! Este projeto é uma **Landing Page moderna e responsiva**, desenvolvida para apresentar minha jornada profissional, minhas habilidades técnicas e os principais projetos em que venho trabalhando.

## 🛠 Tecnologias Utilizadas
Neste projeto foi utilizado uma stack atual para desenvolvimento web, voltado para melhorar meus conhecimentos nestas tecnologias e criar um portfólio moderno e responsivo:

- **[React](https://react.dev/)**: A biblioteca principal na renderização da interface, gerenciando todo o site através de componentes interativos e reaproveitáveis.
- **[TypeScript](https://www.typescriptlang.org/)**: Um superset rígido do JavaScript que adiciona tipagem forte aos arquivos (`.tsx`/`.ts`). Isso blindou todo o projeto, garantindo uma estrutura escalável com zero falhas de execução no navegador.
- **[Vite](https://vitejs.dev/)**: A ferramenta moderna de build utilizada. Extrema velocidade no desenvolvimento e uma compilação poderosa que reduz drasticamente o tamanho do site.
- **[Tailwind CSS (v4)](https://tailwindcss.com/)**: Um dos melhores frameworks de CSS do mercado para estilização baseada em utilitários.
- **[Lucide React](https://lucide.dev/)**: Um forte e minimalista acervo de ícones (`SVG`) para melhorar as referências visuais em todas as seções (Github, Instagram, Terminal, Maleta, Código etc).
